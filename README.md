# VPS Toolbox - 服务器工具箱

Linux VPS 管理工具箱脚本，集成多种常用功能，方便快速管理服务器。

## 快速安装

执行以下命令安装必要依赖：

```bash
apt-get update -y && apt-get install -y curl
```

### 🚀 运行

**通用地址：**
```bash
bash <(curl -fsSL tool.wwwo.eu.cc)
```

**国内加速：**
```bash
bash <(curl -fsSL toolx.wwwo.eu.cc)
```

## 使用说明

### 快捷命令
脚本安装完成后，会自动配置快捷入口，无需再次输入长命令。在终端输入以下字母即可快速启动：
* `m`
* `M`

### 功能模块
本工具箱包含以下核心分类：
* **系统设置**：内核调优、Swap 配置等
* **网络代理**：主流协议一键部署
* **网络检测**：流媒体解锁、三网回程测试
* **Docker管理**：容器环境及面板安装
* **应用商店**：常用建站、实用程序安装
* **证书管理**：SSL 证书申请与续期
* **系统管理**：资源监控、用户管理
* **工具箱合集**：集成其他优秀脚本
* **玩具熊ʕ•ᴥ•ʔ**：趣味功能模块
* **监控通知**：探针及消息通知配置
* **备份恢复**：数据迁移与备份
* **更新卸载**：脚本自身的维护与移除

### 操作流程
1.  **安装**：运行上方安装命令。
2.  **启动**：输入 `m` 打开菜单。
3.  **选择**：根据数字提示输入对应编号执行任务。
4.  **返回/退出**：按回车键返回主菜单，输入 `0` 退出。
